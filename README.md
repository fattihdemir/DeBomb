# DeBomb

**DeBomb**, oyuncunun kelime tahmini yaparak bir şehri kurtarmaya çalıştığı Unity tabanlı bir oyun projesidir.

## Oyun Mantığı
- Oyuncu, verilen ipuçlarına göre doğru kelimeleri tahmin eder.
- Her yanlış tahmin şehrin güvenliğini azaltır.
- Tüm kelimeler doğru tahmin edildiğinde şehir kurtulur ve oyuncu kazanır.

## Proje Yapısı
- **Assets/** : Oyunun sahneleri, scriptleri ve görselleri.
- **ProjectSettings/** : Unity proje ayarları.

## Kurulum
1. Unity Hub üzerinden uygun Unity sürümünü açın.
2. Projeyi klonlayın veya indirin.
3. Unity Hub üzerinden proje klasörünü açın ve sahneleri çalıştırın.
